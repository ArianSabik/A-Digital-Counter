
Digital Counter Using 555 Timer and CD4026 with 7-Segment Display 
Project Introduction:
This project presents a two-digit digital counter (00–99) using a 555 timer IC in astable mode for 
clock generation and two CD4026 decade counter ICs to drive seven-segment displays. The units 
counter feeds the tens counter through its carry output, enabling continuous counting.
Additional features include a pause switch, reset switch, pulse indicator LED, and a BC547-
based LED that lights at every 10 counts (10, 20, etc.). A 100 kΩ potentiometer is used to control 
counting speed by adjusting the clock frequency.
Overall, the project demonstrates key digital electronics concepts like clock generation, 
cascading counters, and display interfacing without a microcontroller, and was tested for stable 
and accurate operation.
Objectives:
1. To design a digital counter capable of counting from 00 to 99. 
2. To generate clock pulses using a 555 timer IC. 
3. To implement decade counting using CD4026 ICs. 
4. To interface seven-segment displays with digital counter circuits. 
5. To understand cascading techniques used in multi-digit counters. 
6. To implement pause and reset control functionality. 
7. To implement adjustable counting speed using a potentiometer. 
8. To design LED indicators for pulse monitoring and decade counting. 
9. To analyze the behavior of sequential digital circuits.
